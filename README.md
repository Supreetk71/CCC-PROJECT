Design and Analysis of Algorithms (DAA)

📖 Project Description

The Smart Diet Planner is an optimization-based application that helps users select the most nutritious combination of food items within a given budget.

This project demonstrates the practical implementation of two important algorithmic techniques:

Greedy Algorithm
Dynamic Programming (0/1 Knapsack Problem)

The system compares both approaches and highlights the difference between approximate and optimal solutions.

🎯 Objectives
Maximize calorie intake within a limited budget
Apply Greedy and Dynamic Programming techniques
Compare efficiency and accuracy of both algorithms
Demonstrate real-world use of DAA concepts
🧠 Algorithms Used
🔹 Greedy Algorithm
Selects food items based on maximum calories per cost ratio
Fast and efficient
May not always give optimal results
🔹 Dynamic Programming (Knapsack)
Considers all possible combinations
Guarantees optimal solution
Slightly higher computational cost
